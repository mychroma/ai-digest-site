테스트: n8n으로 이메일 자동 분류하기

매일 아침 업무 관련 이메일을 하나하나 열어보고 중요한 것만 골라내는 데 시간을 너무 많이 쓰고 계신가요?

⏱ 소요시간

약 30분

📊 난이도

초급

🔧 필요 툴

n8n, Gmail, Claude API

✨ 이렇게 해결할 수 있어요

n8n과 AI를 연결하면 이메일이 오는 순간 자동으로 분류하고 중요한 것만 알림을 보내줘요.

📊 자동화 전 vs 후

😩 자동화 전

매일 아침 30분씩 이메일함을 열어 하나씩 확인하고 중요한 것만 따로 정리

🚀 자동화 후

중요한 이메일만 슬랙으로 자동 알림, 나머지는 자동 분류 완료

🛠 사전 준비

n8nGmailClaude API
  • n8n 계정 (무료로 가입 가능)
  • Gmail 계정
  • Claude API 키 (anthropic.com에서 발급)

📋 단계별 구현

1
트리거 n8n에서 새 워크플로우 만들기

n8n에 로그인한 후 오른쪽 상단의 '+ New Workflow' 버튼을 클릭해요. 빈 캔버스가 나타나면 왼쪽 패널에서 'Gmail' 을 검색해서 'Gmail Trigger' 노드를 드래그해서 가져다 놓아요. 이 노드가 새 이메일이 올 때마다 자동으로 시작하는 역할을 해요.

💡 Gmail 연동 시 구글 계정 로그인 팝업이 뜨면 허용을 눌러주세요.
2
AI AI에게 이메일 분류 맡기기

캔버스에 'HTTP Request' 노드를 추가하고 Claude API 주소를 입력해요. Body에는 이메일 제목과 내용을 넣고 '이 이메일의 중요도를 긴급/일반/참고 중 하나로 분류해줘'라고 지시문을 작성해요.

💡 API 키는 절대 코드에 직접 쓰지 말고 n8n의 Credentials 기능을 사용해서 안전하게 저장하세요.
3
조건 결과에 따라 다르게 처리하기

'IF' 노드를 추가해서 AI가 '긴급'이라고 답한 경우와 아닌 경우를 나눠요. 긴급인 경우에는 Slack 노드를 연결해서 즉시 알림을 보내고, 나머지는 Gmail 노드로 라벨만 붙여요.

✅ 완성하면 이렇게 돼요

이제 이메일이 오면 AI가 자동으로 읽고 분류해요. 긴급한 것만 슬랙으로 알림이 오고, 나머지는 자동으로 정리돼요. 아침에 이메일함을 일일이 확인할 필요가 없어져요.
