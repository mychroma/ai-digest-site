Claude Cowork로 카드뉴스 자동 생성하기

SNS 카드뉴스, 매번 만들 때마다 시간이 너무 많이 걸리지 않나요? 캔바 열고, 텍스트 정리하고, 디자인 맞추고, 이미지 찾고... 하나 만드는 데 1~2시간은 기본이잖아요. 특히 정기적으로 올려야 하는 콘텐츠라면 이게 정말 큰 부담이 됩니다.

약 30분

📊 난이도

초급

🔧 필요 툴

Claude Cowork, Figma, Claude API

✨ 이렇게 해결할 수 있어요

Claude Cowork의 Figma 연동 기능을 쓰면, 미리 만들어둔 카드뉴스 템플릿에 AI가 자동으로 내용을 채워줘요. 주제만 던져주면 텍스트 작성부터 슬롯 채우기까지 자동으로 처리되는 방식이에요.

📋 단계별 구현

1
설정 Figma에서 카드뉴스 템플릿 만들기

Figma에 로그인한 후 New design file을 클릭해 새 파일을 만들어요. 왼쪽 상단 + 버튼을 눌러 Frame을 추가하고, 사이즈를 1080x1080으로 설정하면 인스타그램 정사각형 비율이 됩니다. 여기에 제목 텍스트 박스, 본문 텍스트 박스, 배경 색상을 배치해요. 핵심은 AI가 채울 텍스트 박스에 이름을 명확하게 지정하는 거예요. 텍스트 박스를 클릭하고 오른쪽 레이어 패널에서 이름을 '제목', '본문1', '본문2'처럼 알아보기 쉽게 바꿔주세요.

이미지를 추가하려면 sample_data.json의 steps[0].image에 이미지 URL 또는 파일명을 입력하세요
2
설정 Figma에서 공유 링크 복사하기

템플릿 완성 후 오른쪽 상단 Share 버튼을 클릭해요. 팝업창에서 'Anyone with the link'를 'can edit'으로 설정하고 Copy link를 클릭해요. 이 링크가 Claude Cowork와 연결할 때 필요한 주소예요.

이미지를 추가하려면 sample_data.json의 steps[1].image에 이미지 URL 또는 파일명을 입력하세요
3
연동 Claude Cowork에서 Figma 연결하기

claude.ai에 로그인한 후 왼쪽 메뉴에서 Cowork를 클릭해요. 새 Cowork 세션을 시작하면 채팅창이 열려요. 여기서 도구 연결 버튼을 찾아 Figma를 선택해요. Figma 계정으로 로그인하는 팝업이 뜨면 허용을 눌러주세요.

이미지를 추가하려면 sample_data.json의 steps[2].image에 이미지 URL 또는 파일명을 입력하세요
4
AI 템플릿 링크와 주제 전달하기

Claude Cowork 채팅창에 아까 복사한 Figma 링크와 함께 만들고 싶은 카드뉴스 주제를 입력해요. 슬라이드 장 수, 타깃 독자, 원하는 톤까지 함께 알려주면 더 정확하게 작업해줘요. Claude가 Figma 파일을 읽고 레이어 구조를 파악한 다음 각 텍스트 박스에 내용을 채우기 시작해요.

이미지를 추가하려면 sample_data.json의 steps[3].image에 이미지 URL 또는 파일명을 입력하세요
5
테스트 결과물 확인 및 수정 요청하기

Claude가 작업을 완료하면 Figma 파일이 자동으로 업데이트돼요. Figma를 새로고침하면 각 텍스트 박스에 내용이 채워진 걸 확인할 수 있어요. 마음에 안 드는 부분이 있으면 채팅창에 바로 수정 요청을 하면 돼요.

이미지를 추가하려면 sample_data.json의 steps[4].image에 이미지 URL 또는 파일명을 입력하세요

✅ 완성하면 이렇게 돼요

주제 하나를 던지면 10분 안에 완성된 카드뉴스 5장이 Figma에 준비돼요. 기존에 장당 20~30분 걸리던 작업이 전체 합쳐서 10분으로 줄어요.

💰 비용 정보

Claude Cowork는 현재 베타로 무료 사용 가능해요. Figma는 무료 플랜으로도 충분하고, Claude API는 사용량에 따라 과금되지만 카드뉴스 몇 장 만드는 수준이면 한 달에 $1~2 수준이에요.

👥 실제 활용 사례

  • 마케터라면 매주 브랜드 SNS에 올릴 정보성 콘텐츠를 이 방식으로 빠르게 만들 수 있어요.
  • 교육 담당자라면 사내 교육 자료나 공지사항을 카드뉴스 형태로 빠르게 제작해 공유할 수 있어요.
  • 1인 창업자나 프리랜서라면 포트폴리오 소개, 서비스 안내 카드뉴스를 직접 빠르게 만들어 비용을 절약할 수 있어요.

⚠️ 자주 하는 실수

문제 상황

Claude가 Figma 레이어를 못 찾겠다고 해요.

해결법

레이어 이름이 한글이거나 특수문자가 포함된 경우예요. 영문 소문자로 title, body처럼 단순하게 바꿔주세요.

문제 상황

내용은 채워졌는데 텍스트가 박스를 넘쳐요.

해결법

Figma에서 해당 텍스트 박스를 선택하고 Text를 Auto height로 설정하면 자동 조정돼요.

문제 상황

카드뉴스 내용이 너무 딱딱하고 AI 같아요.

해결법

프롬프트에 톤을 명시해주세요. '20대 직장인 친구에게 말하듯이'처럼 구체적인 스타일을 지정하면 훨씬 자연스러워져요.

💡 이런 상황에도 써보세요

  • 회사 주간 보고서 템플릿을 만들어두면, 매주 데이터만 넣어서 보고서를 자동 완성할 수 있어요.
  • 제품 소개 슬라이드 템플릿을 만들어두면 신제품 출시할 때마다 발표 자료가 자동 완성돼요.
  • 이벤트 배너 템플릿을 여러 사이즈로 만들어두면 한 번 요청으로 인스타, 페이스북, 유튜브 썸네일용 배너를 동시에 만들 수 있어요.
